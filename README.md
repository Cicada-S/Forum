## 校园论坛


### 前言

校园论坛小程序 主要分区:

娱乐八卦  二手市场  表白墙  失物招领  学习交流


### 项目功能

- [x] 登录
- [x] 获取用户信息
- [x] 更新用户信息
- [x] 发布帖子
- [x] 帖子图片/视频
- [x] 帖子分区
- [x] 搜索
- [x] 资讯/轮播图
- [x] 我的帖子
- [x] 关于我们
- [x] 点赞收藏
- [x] 评论(父评/子评/回复)
- [x] 转发


### 项目构建

```sh
npm install

```

### 数据表

| 表名 | 备注   |
| ---- | ------ |
| User | 用户表 |
| Post | 帖子表 |
| PostMedia | 帖子视频图片表 |
| FatherComment | 父评论表 |
| SonComment | 子评论表 |
| AgreeCollect | 点赞收藏表 |
| Information | 资讯表 |
| Reward | 打赏表 |
